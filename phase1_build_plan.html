M4 Data tracker 1 file · data layer · ~3 hrs
Silent observer. Records every play, skip, and like into IndexedDB. Invisible to the user, but the smarter Phase 2 features only work if this has been running and accumulating signal.
What it tracks
MutationObserver → detects now-playing changes
videoId, title, artist, timestamp, duration
Skip detection (< 30s elapsed → skip signal)
Like / dislike button click events
Incremental updates to song_scores store
